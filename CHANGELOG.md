# Changelog

All notable changes to the "Selected Code Explainer (AI)" extension are documented here.

## [1.0.0] - 2026-07-06

### Added
- Initial release.
- Explain any selected code snippet, in any language, using OpenRouter (Qwen, DeepSeek, Llama, GPT-OSS, and more).
- Accessible via Command Palette, right-click context menu, editor title icon, and keybinding (`Ctrl+Alt+E` / `Cmd+Alt+E`).
- Native hover popup (default) shows a one-sentence explanation right next to the selection; a secondary command opens a detailed side panel.
- Rich, theme-aware webview panel with Markdown rendering, copy button, and regenerate button.
- Configurable explanation detail level (brief / standard / detailed), response language, model, temperature, and max output tokens.
- Status bar quick-access button.
- Friendly, specific error messages for invalid keys, out-of-credit errors, rate limits, and network issues.
